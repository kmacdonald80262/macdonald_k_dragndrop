# Interactive Puzzle Game


### Drag and Drop Puzzle Pieces

This is an interactive puzzle building game where the user drags the puzzle pieces beofre dropping them
in an order to form the picture.

Prerequisites: All you need is an internet browser, but this site works best on [Google Chrome](https://www.google.com/chrome/)

### Installing

- git init

- Clone this repository

- git clone https://github.com/kmacdonald80262/macdonald_k_dragndrop.git

- Go into the repository

- cd macdonald_k_dragndrop

Then you can play this game!
